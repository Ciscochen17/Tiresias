# Tiresias
> Tiresias(提瑞西阿斯)是古希腊神话中的人物，他虽然双目失明，却拥有超凡的预言能力和对真相的深刻理解。这种“盲眼先知”的形象寓意着真正的智慧和远见往往超越了肉眼所见的表象，来自于更深层次的精神或神性启示。本项目最终目的是结合LLM准确预测大A的市场动向（从单个股到多个股的集合），与该人物寓意契合。
